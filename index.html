Мобильное приложение "Такси"

При разработке данного проекта использавалась база Firebase для синхронизации местоположения, СУБД PostgreSQL для хранения всех данных. И RestAPI написанный на Python.

Данное приложение разрабатывалось на заказ для удобного доступа к электронной библиотеки. Для данного приложения также была написана серверная часть на Python с использованием фреймворка Flask. Также реализована синхронизация информации о скачанных книгах с Google Sheets.

Данное прложение является ИДЗ по ТИС. Для его реализации была спроектирована и реализована БД в СУБД PostgreSQL, а для обмена данными между БД и приложением была написана серверная часть на Python (Flask). Данный проект еще не завершен.

Десктопное приложение “Морской бой”

Данное приложение написано на языке C# с использованием WPF. В данной игре предусмотрена возможность как игры против компьютера (запрограммирован алгоритм игры), так и против других игроков по сети и интернету (была написана отдельная серверная часть).
